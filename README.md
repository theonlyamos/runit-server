# RunIt-Server

## Installation
Follow the commands below to install the package
```shell
pip install runit-server
```

## Usage
Follow the instructions below to use the package

### Check Version

```shell
runit-server --version|-v
```

* Run the sever

```shell
runit-server
```

* Setup
<p>Run the setup command before starting the server</p>

```shell
runit-server setup
```
